Palettizer

HEX Code Palette Generator

Purpose:

Originally conceived to allow a company with multiple brands, and therefore multiple style-guides, to create a multi-brand color palette for report generation and dashboard design.

Given a series of HEX keys, this app allows a user to generate a custom palette, which can then be used to check overall asthetics as well as colorblind accessibility. Auto-generates JSON or XML for creation of Power BI themes and Tableau color sets respectively.

How to use:

Check out the video demo
